# StudentMangerSystem
学生信息管理系统是一个基于Android手机上运行的管理学生信息的安卓应用， 通过在Android平台下设计各个Activity布局和控件与结合SQLite数据库使用Java进行开发的一款Android端的学生信息管理系统，主要包括学生信息添加、学生信息维护、学生信息查询、学生成绩添加、学生成绩维护、学生成绩查询、系统管理、使用帮助、退出软件等功能
